+++++++++++++++++++++
cptrace Python module
+++++++++++++++++++++

Python binding for ptrace written in C.

Example
=======

Dummy example: ::

    >>> import cptrace
    >>> cptrace.ptrace(1, 1)
    Traceback (most recent call last):
      File "<stdin>", line 1, in <module>
    ValueError: ptrace(request=1, pid=1, 0x(nil), 0x(nil)) error #1: Operation not permitted
